# multisig-tool

windows版本打包命令
CGO_ENABLED=0 GOOS=windows GOARCH=amd64 go build main.go

linux版本打包命令
CGO_ENABLED=0 GOOS=linux GOARCH=amd64 go build main.go

## Linux 

./multisig-tool help

## macOS

./multisig-tool help

## windows

In the command line (CMD) tool, execute the file, example:

D://multisig-tool.exe help
